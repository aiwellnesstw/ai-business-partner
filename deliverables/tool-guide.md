## 工具一：Google Maps 名單爬蟲

### 這個工具做什麼？

每次執行，自動在 Google Maps 上搜尋你設定的「地區 + 行業關鍵字」，
把所有出現的商家名稱、電話（手機優先）、地址存進 CSV 檔案。

### 環境需求

- 電腦：Windows 10/11（Mac 也可以）
- Node.js：v18 以上（下載：nodejs.org）
- 課程提供的腳本檔案：`scrape_gmaps.js`

### 安裝步驟

1. 安裝 Node.js（下載後一路按下一步即可）
2. 打開「命令提示字元」（在搜尋列輸入 cmd）
3. 輸入以下指令，安裝必要套件：
   ```
   cd 你的腳本所在資料夾路徑
   npm install
   ```

### 使用方式

**Step 1：設定搜尋條件**

用記事本打開 `scrape_gmaps.js`，找到這一段修改：

```javascript
const KEYWORDS = ['鋼鐵廠', '製造業', '食品加工'];  // 改成你的行業關鍵字
const AREAS    = ['台中市', '彰化縣'];               // 改成你要的地區
```

**Step 2：執行腳本**

回到命令提示字元，輸入：
```
node scrape_gmaps.js
```

**Step 3：取得結果**

執行完成後，同資料夾內會出現 `gmaps_results_日期.csv`，
用 Excel 開啟即可看到名單。

### 常見問題

**Q：跑一半停了怎麼辦？**  
A：直接重新執行，腳本會自動跳過已抓過的項目，從中斷點繼續。

**Q：抓到市話怎麼辦？**  
A：腳本預設優先抓手機號碼，市話會在欄位中標記，方便你篩選。

**Q：一次能抓多少？**  
A：視網路速度，通常一個關鍵字 + 地區組合約 30-100 筆，速度約 5-10 分鐘。

## 工具二：104 職缺名單爬蟲

### 這個工具做什麼？

搜尋 104 上特定職缺（例如：業務、採購、主任），抓出有在徵人的公司名稱、
公司規模、電話（如有），存進 CSV 方便你聯繫。

### 環境需求

同工具一（Node.js 環境）

### 使用方式

**Step 1：設定搜尋條件**

打開 `scrape_104.js`，修改以下欄位：

```javascript
const JOB_KEYWORDS = ['業務'];         // 職缺關鍵字
const AREA_CODE    = '6001001000';     // 地區代碼（台北=6001001000，見附表）
const MAX_PAGES    = 10;              // 最多抓幾頁（1頁約30筆）
```

| 台南市 | 6001011000 |
| 高雄市 | 6001014000 |

**Step 2：執行**

**Step 3：取得結果**

## 工具三：標案即時通知 Bot

### 這個工具做什麼？

每天早上自動掃描政府電子採購網（eTender），
只要有符合你關鍵字的新公告，立刻推送到你的 Telegram。

### 環境需求

- VPS 主機 或 電腦（需長時間開著，建議用 VPS）
- Telegram 帳號
- 課程提供的 Bot 腳本

### 設定步驟

**Step 1：建立 Telegram Bot**

1. 在 Telegram 搜尋 `@BotFather`
2. 輸入 `/newbot`，照指示設定名稱
3. 記下給你的 **Bot Token**（格式：`1234567890:AABBC...`）
4. 開啟你建立的 Bot，取得你的 **Chat ID**（輸入 `/start` 後到這個網址查：`https://api.telegram.org/bot你的TOKEN/getUpdates`）

**Step 2：設定關鍵字與 Bot Token**

打開 `config.json`，填入你的資訊：

```json
{
  "telegram_token": "你的 Bot Token",
  "chat_id": "你的 Chat ID",
  "keywords": ["無人機", "資訊安全", "AI", "設備採購"],
  "min_budget": 100000
}
```

**Step 3：測試執行**
```
node tendering_bot.js --test
```
如果 Telegram 收到測試訊息，代表設定成功。

**Step 4：設定每日自動執行**

Windows 工作排程器：
1. 搜尋「工作排程器」
2. 建立基本工作 → 每日 → 設定時間（建議早上 8:00）
3. 動作：啟動程式 → 填入 `node` 和腳本路徑

### 通知格式範例

```
🔔 新標案通知

機關：台中市政府
標案名稱：AI 智慧系統採購案
預算金額：NT$3,500,000
截止日期：2026/07/15
公告連結：https://...

關鍵字命中：AI
```

## 工具四：Claude AI 助理使用指南

### 怎麼開始使用

1. 前往 claude.ai 或安裝 Claude 手機 App
2. 免費版可完成大部分功能；進階功能建議升級 Claude Pro（約 NT$620/月）

### 搭配 Prompt 庫使用

1. 打開 `prompt-library.md`（課程附件）
2. 找到對應情境的 Prompt（例如 P-11 提案架構生成）
3. 複製 Prompt，填入 `【】` 內的資訊
4. 貼進 Claude 對話框，送出

### 讓 Claude 記住你的背景（設定 Project）

1. Claude 網頁版 → 左側 Projects → 建立新 Project
2. 在 Project Instructions 填入：
   ```
   你是我的業務助理。以下是我的背景資訊：
   - 我的姓名：Justin
   - 我的公司/產品：【填入你的資訊】
   - 主要客群：【行業/職稱】
   - 我的銷售風格：【例如：直接、有溫度、不強迫推銷】
   
   每次幫我寫文案或話術時，都要符合我的風格，不要太像 AI 生成的罐頭文字。
   ```
3. 之後在這個 Project 裡的對話，Claude 都會記住你的背景。

## 名單去重：避免重複聯絡同一家公司

如果你同時使用 Google Maps 和 104 爬蟲，可能會有重複的名單。

建議做法：
1. 把兩個 CSV 都開在 Excel
2. 選擇所有資料 → 資料 → 移除重複項 → 勾選「電話」欄位

或者直接告訴我，我幫你寫一個自動去重的腳本。
