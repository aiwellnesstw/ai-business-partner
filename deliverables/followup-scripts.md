# AI 業務夥伴 — 客戶跟進話術模板（10 種情境）

> 使用說明：根據情境選擇對應模板，填入 `【】` 內的資訊後直接發送，或貼給 Claude 請他根據你的語氣調整。

---

## 情境 1：傳訊息後已讀不回（3 天內）

**適用時機：** 傳出提案/報價後，對方已讀但超過 3 天沒回應。

**模板 A（輕鬆版）：**
```
【對方姓名】你好，想說確認一下你有沒有收到上次傳的資料？
如果有任何問題或想多了解哪部分，隨時跟我說 😊
```

**模板 B（提供新資訊版）：**
```
【對方姓名】你好！
上次傳的【產品/報價】不知道你看了嗎？
剛好這週有個【新消息/案例/優惠】想讓你參考，傳給你看看。
有問題的話我們可以約個時間聊 10 分鐘。
```

---

## 情境 2：已讀不回第二次（換策略）

**適用時機：** 跟了兩次都沒有回應，需要換方向切入。

**模板：**
```
【對方姓名】，我知道你最近可能很忙。
不一定要現在決定，只是想讓你知道有問題隨時都可以找我。

對了，你們最近有沒有在規劃【相關業務/採購】？
我這裡剛好有一些【行業】的最新資訊，不確定對你有沒有幫助，
有興趣的話可以聊幾分鐘？
```

---

## 情境 3：對方說「再考慮看看」後的跟進

**適用時機：** 客戶沒有明確拒絕，說要考慮，過了 1-2 週無聲。

**模板 A（直接問版）：**
```
【對方姓名】你好，上次聊到【產品/方案】，不知道目前評估到哪個階段了？
如果有什麼顧慮或需要補充的資料，我可以協助你。
```

**模板 B（製造誘因版）：**
```
【對方姓名】你好！
想讓你知道我們這個月的【早鳥優惠/合作名額】還剩最後幾個，
想說先讓你知道，方便你做決定。
有什麼問題直接告訴我，我們來想辦法 😊
```

---

## 情境 4：直接被拒絕後的回應

**適用時機：** 對方明確說「不需要」「沒有預算」「不感興趣」。

**模板 A（優雅收尾，留後路）：**
```
了解，謝謝你的時間！
如果之後有需要，或是身邊有朋友/同業有類似需求，歡迎再找我。
祝你工作順利 🙂
```

**模板 B（問原因版，適合已有一定關係的客戶）：**
```
謝謝你告訴我，請問方便讓我了解一下是哪個部分讓你猶豫嗎？
我想確認是我沒有說清楚，還是真的不適合你們現在的需求，
這樣我以後也可以做得更好。
```

---

## 情境 5：客戶說「太貴了」

**適用時機：** 對方對產品感興趣，但覺得報價超出預期。

**模板 A（重新框架價值）：**
```
我理解價格是很重要的考量。
讓我換個角度說：如果【產品/服務】能幫你【具體效益，例如：每個月省下 XX 小時/多開發 X 個客戶】，
這樣算下來，成本大概是【計算結果】。

你覺得這樣划算嗎？我們可以聊聊你目前最在意的是哪部分。
```

**模板 B（彈性方案）：**
```
我知道預算是個重要考量，這樣吧——
讓我了解一下你們目前的預算範圍，看我能不能調整方案，
找到一個對你來說更舒服的起點。
```

## 情境 6：節日 / 時事暖身（重新建立連結）

**適用時機：** 久沒聯絡的客戶，借節日或相關時事自然重新接觸。

**模板（農曆新年版）：**
```
【對方姓名】新年快樂！
祝你今年業績長紅、出入平安 🧧
有空再來聊聊，期待今年有機會合作！
```

**模板（產業相關事件版）：**
```
【對方姓名】你好，看到最近【產業新聞/政策】，
想到你們之前在做【相關業務】，不知道這對你們有沒有影響？
如果有需要，我這邊剛好有一些相關資訊可以分享給你。
```

---

## 情境 7：成交後請客戶轉介紹

**適用時機：** 客戶用了一段時間，反應正面，適合開口請他介紹。

**模板：**
```
【對方姓名】，用了這段時間，不知道整體感覺如何？

如果還不錯的話，有沒有身邊的朋友或同業，也有類似的需求？
我不是要你幫我推銷，只是如果你覺得對他們有幫助，
介紹給我認識，我一樣會好好服務他們。

當然，如果你方便的話，一句話推薦就很夠了 😊
```

---

## 情境 8：合約到期前的續約提醒

**適用時機：** 合約還有 1-2 個月到期，需要提前溝通續約。

**模板：**
```
【對方姓名】你好，想提醒你我們的合約大約【月份】到期。

這段時間合作下來，有什麼想調整或加強的地方嗎？
我這邊整理了一些使用狀況，可以找時間跟你報告一下，
順便聊聊後續的方向。

你這週什麼時候方便？
```

---

## 情境 9：長期沒聯絡的老客戶重新接觸

**適用時機：** 超過半年沒有往來，想重新建立關係。

**模板 A（直接切入版）：**
```
【對方姓名】好久不見！
最近剛好有個跟你們行業相關的【新方案/案例/資訊】，
第一個想到你，看你有沒有興趣了解一下？
```

**模板 B（關心版，適合關係比較好的）：**
```
【對方姓名】！很久沒聯絡，你還好嗎？
最近看到你們公司【有新消息/在某個場合看到你】，
想說打個招呼，有空的話可以聊聊聚聚。
```

---

## 情境 10：客戶抱怨 / 問題處理後的跟進

**適用時機：** 之前有出現問題或客訴，處理完後跟進確認。

**模板：**
```
【對方姓名】你好，上次【問題描述】的事情，這邊已經【處理結果】，
請問你那邊確認都 OK 了嗎？

如果有任何問題，直接找我，我來處理。
非常感謝你的包容，我們會繼續注意這部分。
```

---

## 使用小技巧

1. **不要直接 copy 貼上** — 稍微加入你自己的語氣，看起來更自然
2. **搭配 Prompt 庫使用** — 情況複雜時，把這些模板貼給 Claude，請他根據你的具體情況調整
3. **建立自己的版本** — 用了一段時間後，把效果好的回覆存起來，建立你自己的專屬話術庫
4. **時機比話術更重要** — 對的時間點發出去，勝過最完美的文字
